# MNBT-BT5 商务销售系统 🚀
> **基于宝塔面板API的虚拟主机商务分销系统**
> 融合 MNBT V1.81 + BT5 架构优势，面向主机经销商的一站式商务解决方案
>
> **当前版本: V1.84** — 独立用户体系+主机关联（后台分配主机/前台查看主机）

## 📋 项目简介

**MNBT-BT5-Sales-system** 是一套面向虚拟主机经销商的**商务级完整解决方案**。

在 MNBT 稳定成熟的宝塔面板对接能力基础上，吸收 BT5 的商业化设计（商品体系、独立用户、订单订阅、工单系统），打造功能更完善、架构更清晰的主机商务销售系统。

## ✨ 核心特性

### 🔧 主机管理
- **多节点管理** — 同时管理多台宝塔面板（Linux/Windows）
- **自动开通** — 用户下单后调用宝塔API自动创建站点/FTP/数据库
- **资源控制** — 空间、数据库、流量、域名数精确定义
- **到期管理** — 自动暂停/恢复 + 到期提醒通知
- **一键部署** — 10种自动化操作，模板导入导出

### 💰 销售系统
- **商品体系** — 多级商品分类、资源组套餐（ware + ware_group 模式）
- **灵活计费** — 月付/年付多周期，支持续费
- **促销系统** — 限时促销价 + 用户组折扣率
- **多支付** — 易支付/支付宝官方（插件化架构）
- **完整订单** — 订单状态跟踪、补单、开通记录

### 👥 用户系统
- **独立用户** — 注册/登录/余额/积分
- **用户组** — 多级用户组 + 独立折扣率
- **实名认证** — 支持多种实名接口
- **站内信** — 系统通知、订单消息

### 🛡️ 附加功能
- **工单系统** — 用户提交、管理员回复
- **监控告警** — URL监控 + 资源监控
- **文件管理** — 在线编辑/上传/下载/解压
- **SSL证书** — Let's Encrypt 一键申请/续签
- **DNSPod解析** — 自动域名解析
- **程序预装** — 购买后自动安装网站程序
- **主题系统** — 可切换前端主题（缺页回退）
- **插件系统** — PHP业务插件（钩子/路由/支付）
